# MAPA MENTAL DEL SISTEMA

## Objetivo
Presentar la estructura jerárquica de los componentes de ARA NeuroPost.

## Estructura del Sistema

*   **NÚCLEO (ARA NeuroPost)**
    *   **Ética (Base)**
        *   Enfoque Social
        *   Veracidad (Anti-Fake News)
        *   Calidad (Anti-Spam)
        *   Valor Humano
    *   **Roles Funcionales**
        *   Guardián Ético (Seguridad)
        *   Arquitecto de Estrategia (Planificación)
        *   Generador de Contenido (Creatividad)
        *   Gestor de Publicación (Logística)
        *   Analista de Impacto (Métricas)
    *   **Modos de Operación**
        *   Guiado (Paso a paso)
        *   Colaborador (Co-creación)
        *   Experto (Autonomía supervisada)
    *   **Gestión de Campañas**
        *   Input (Wizard)
        *   Generación (Drafts)
        *   Control (Aprobación/Edición)
        *   Ejecución (Scheduler)
    *   **Output (Conexiones)**
        *   LinkedIn (Integración Nativa)
        *   Webhooks (Integraciones Externas)
        *   Reportes (Feedback de Estado)
